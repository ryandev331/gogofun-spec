    # 基本入門：廣告建立/廣告架構/廣告目標設定

    # 官方說明

    # 1. 廣告建立 官方說明

    👉廣告管理員介面:按下建立button

    切換至引導式建立👇

    👉引導式示建立:依序完成行銷活動、廣告組合、廣告等設定

    # 2. 廣告架構 官方說明

    • 行銷活動：廣告的基礎，設定廣告目標，可包含多個廣告組合
    • 廣告組合：廣告的刊登方式，設定目標設定(廣告受眾)、預算、排程、出價、版位等，可包含多則廣告
    • 廣告：廣告受眾看到的內容，設定文字(廣告文案)、圖像、影片、button等項目

    # 3. 廣告目標設定：3大類（11小類）官方說明

    • 品牌認知：品牌知名度、觸及人數
    • 觸動考量：流量、互動、應用程式安裝、觀看影片、開發潛在顧客、發送訊息
    • 轉換行動：轉換次數、目錄銷售、來店客流量

    # 建立目錄銷售廣告（適用有建立商品目錄的業主）：
    行銷活動設定/廣告帳號/廣告組合/廣告

    # 官方說明

    # 1.行銷活動設定：行銷目標設定"目錄銷售"，並完成以下設定

    • 行銷活動名稱：自定義名稱
    • 商品目錄：選擇擁有的商品目錄
    • 建立分組測試：選擇測試變數（廣告創意、投遞最佳化、版位） 官方說明
    • 行銷活動預算最佳化：行銷活動預算、行銷活動出價策略 官方說明

    # 2.廣告帳號：照預設值直接按右下角繼續~

    # 3.廣告組合：

    廣告組合名稱：自定義的廣告組合名稱

    廣告目錄組合 & 優惠：選擇想要促銷的商品目錄

    廣告受眾：依照想觸及的目標群眾去設定

    版位：自動版位或自行選擇喜歡的版位

    預算和排程：

    # 4.廣告：

    廣告名稱：自定義的廣告名稱

    身分：選擇廣告中顯示的識別身分（FB/IG）

    格式：廣告格式（輪播/單一圖像）

    格式：廣告創意：設定廣告顯示的文案、輪播商品(簡介/目錄/個人檔案)、連結、行動呼籲

    格式：廣告追蹤（fb pixel/動態參數追蹤）
