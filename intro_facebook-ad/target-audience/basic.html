# 地點 官方說明

  1. 位於此地點的所有人（預設）：此選項可讓您觸及居住地或最近出現地點在所選區域內的用戶。例如，您可能想向某個地點中的所有人，顯示大型展覽或知名體育活動的廣告。

  2. 居住在此地點的人：此選項可讓您觸及居住地位在所選區域內的用戶。例如，您可能想向居住在倫敦的用戶顯示當地零售店的廣告。

  3. 最近在此地點的人：此選項可讓您觸及最近出現地點在所選區域內的用戶。例如，您可能想向最近在所選地點的用戶，顯示限時特價活動的廣告。

  4. 在此地點旅行的人：此選項包括根據我們收集到的行動裝置和連線資訊，顯示為在所選區域旅遊，且與其本國住處距離超過 125 英里的用戶。例如，您可能想向最近在某地點旅行的用戶，顯示租車服務的廣告。
